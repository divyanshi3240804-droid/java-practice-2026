✅ Day 1 – Basics & Variables
🔹 Java Structure (Very Important)
Every Java program follows this structure:
Java
Copy code
public class Main {

    public static void main(String[] args) {

        // code goes here

    }
}
Rules:
File name must match class name (Main.java → class Main)
Everything must be inside main()
Java is case-sensitive

1️⃣ int → Whole numbers

Java

3️⃣ String → Text (Needs double quotes "")
Java
Copy code

4️⃣ boolean → true / false
Java
Copy code
boolean isStudent = true;

Java

Semicolon ; is compulsory

Add numbers

✅ Day 2 – if / else (Decision Making)
🔹 Basic Structure
Java
Copy code
if (condition) {
    // runs if true
} else {
    // runs if false
}
🔹 Example
Java
Copy code
int marks = 75;

if (marks > 40) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}
🔹 Important Symbols
>  greater than
<  less than
>= greater than or equal
<= less than or equal
== equal to
!= not equal
🔹 else if (Multiple Conditions)
Java
Copy code
if (marks >= 90) {
    System.out.println("Excellent");
} else if (marks >= 40) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}
Important:
Order matters
Java checks from top to bottom
First true condition runs

======================
Day 4 – Loops
======================

1️⃣ What is a Loop?
- Repeat code automatically
- Avoid writing same code multiple times

2️⃣ Types of Loops
- For Loop: fixed number of repetitions
- While Loop: repeat while condition true
- Do-While Loop: runs at least once

3️⃣ Syntax Examples

For Loop:
for(int i=1; i<=5; i++) {
    System.out.println(i);
}

While Loop:
int i = 1;
while(i <= 5) {
    System.out.println(i);
    i++;
}

Do-While Loop:
int i = 1;
do {
    System.out.println(i);
    i++;
} while(i <= 5);

4️⃣ Example – Sum of Numbers (Interactive)
Scanner sc = new Scanner(System.in);
int n = sc.nextInt();
int sum = 0;
for(int i=1; i<=n; i++) {
    int num = sc.nextInt();
    sum += num;
}
System.out.println("Sum: " + sum);

5️⃣ Key Points
- Loops save time & code
- i++ → increment counter
- sum += num → running total
- For known repetitions → for
- For condition → while
- Run at least once → do-while
